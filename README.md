# travel

> travel project

### 多页应用
1. 页面跳转返回html
2. 优点 ： 首屏时间快（只经过了一次http请求），SEO（搜索引擎可以识别html中的内容）效果好
3. 缺点： 页面切换慢
### 单页应用
1. vue应用是单页面应用
2. 页面跳转靠js渲染
3. 优点：页面切换块
4. 缺点：首屏时间慢，SEO差
